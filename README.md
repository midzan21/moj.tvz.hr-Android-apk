Moj TVZ za Android
======================

Android aplikacija za Intranet portal Tehničkog veleučilišta u Zagrebu. 

# Download link

Amazon App Store - http://bit.ly/MojTVZAndroid

Android File Host - http://bit.ly/MojTVZAndroidv4AFHMirror
